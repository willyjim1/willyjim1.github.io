# Unblocked 660

Unblocked 660 is a web-based platform that provides access to a variety of games that can be played without restrictions. This project aims to create a user-friendly interface for game enthusiasts.

## Project Structure

```
unblocked-660
├── src
│   ├── index.html        # Main landing page of the website
│   ├── games.html       # Page displaying available games
│   ├── styles
│   │   └── main.css     # Styles for the website
│   └── scripts
│       └── main.js      # JavaScript functionality for the website
├── package.json          # Configuration file for npm
├── .gitignore            # Files and directories to ignore by Git
└── README.md             # Documentation for the project
```

## Features

- **User-Friendly Interface**: Easy navigation between the main page and games page.
- **Responsive Design**: The website is designed to be accessible on various devices.
- **Interactive Elements**: JavaScript functionality to enhance user experience.

## Setup Instructions

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```
   cd unblocked-660
   ```

3. Install dependencies:
   ```
   npm install
   ```

4. Open the `index.html` file in your browser to view the website.
